---
description: "使用 Qwen CLI 获取技术咨询并制定行动计划"
command: "qwen -p $ARGUMENTS"
---

执行以下Bash命令：

```bash
qwen -p $ARGUMENTS
```

分析其回复，然后提供：

**核心要点**
总结 Qwen 回复中的核心观点和关键信息

**融合思考**
基于 Qwen 的回复，结合自身分析，形成的独立思路、步骤、方案或内容

**建议**
根据融合思考的自信程度，选择以下任一方式：
- 如果方案成熟且自信：建议按上述方案执行
- 如果需要进一步验证：建议先测试上述方案
- 如果存在不确定性：建议继续向 Qwen 咨询 【具体问题】
- 如果需要更多信息：建议先收集 【具体信息】 再实施
